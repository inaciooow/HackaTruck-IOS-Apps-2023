# ⚡ Harry Potter Characters App

Aplicativo desenvolvido durante o HackaTruck com foco em **consumo de API REST e manipulação de dados em aplicações iOS**.

O app consome uma API pública para exibir personagens do universo Harry Potter, permitindo navegação entre lista e detalhes.

---

## 🚀 Funcionalidades

* Consumo de API REST
* Listagem dinâmica de personagens
* Navegação para tela de detalhes
* Carregamento de imagens com AsyncImage
* Exibição de informações dos personagens

---

## 🛠️ Tecnologias utilizadas

* Swift
* SwiftUI
* URLSession
* JSON / Codable
