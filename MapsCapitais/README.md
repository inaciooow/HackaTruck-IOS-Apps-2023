# 🗺️ Maps Capitais App

Aplicativo desenvolvido durante o HackaTruck com o objetivo de explorar o uso do **MapKit** em aplicações iOS.

O projeto demonstra como trabalhar com mapas interativos, anotações e exibição de informações em tela de detalhe.

---

## 🚀 Funcionalidades

* Exibição de mapa interativo
* Adição de marcadores (MapAnnotation)
* Interação ao clicar nas cidades
* Exibição de detalhes em modal (sheet)
* Estrutura de dados personalizada

---

## 🛠️ Tecnologias utilizadas

* Swift
* SwiftUI
* MapKit
